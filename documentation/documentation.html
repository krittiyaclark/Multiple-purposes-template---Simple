Fonts - Installation

If you want to have exactly the same fronts as Simple you have to add Google Fonts at between <HEAD>

<!-- Google Font -->
<link href='http://fonts.googleapis.com/css?family=Kreon' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Droid+Sans' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Sofia' rel='stylesheet' type='text/css'>

CSS Installation

Include all these CSS files at between <HEAD> for Simple layout and responsive format.

<!-- Included CSS Files (Compressed) -->
<link rel="stylesheet" href="css/foundation.css" />
<link rel="stylesheet" href="css/stylesheet.css" />

00 CSS Installation For Color Skins

We provided 5 color skins. You can easily put CSS stylesheet files from CSS folder (green.css, blue.css, red.css, yellow.css, brown.css) and put it at between <HEAD>. For example, GREEN color. You can grab the green.css and pointed to that folder.

<link rel="stylesheet" href="css/green.css" />

01 CSS Installation For Shortcodes

Most Shortcodes are in the shortcodes.css files except Accordions and Buttons are in the foundation.css files so you need to include it at between <HEAD>.

<link rel="stylesheet" href="css/shortcodes.css" />

jQuery Installation

Include jQuery Modernizr at between <HEAD>

<!-- Included JS Files (Compressed) -->
<script src="js/vendor/custom.modernizr.js"></script>

ICONS Installation

Include Font Awesome and Fontello at between <HEAD>

<!-- Icons -->
<link rel="stylesheet" href="icons/fontello/css/fontello.css">
<link rel="stylesheet" href="icons/fontello/css/animation.css"><!--[if IE 7]>
<link rel="stylesheet" href="icons/fontello/css/fontello-ie7.css"><![endif]-->

<link rel="stylesheet" href="icons/font-awesome/css/font-awesome.min.css"><!--[if IE 7]>
<link rel="stylesheet" href="icons/font-awesome/css/font-awesome-ie7.min.css"><![endif]-->

CSS Files

foundation.css - This file contains all of the specific stylings for the page.
stylesheet.css - This file contains all of the custom stylings for the page.
You can edit all these files to make changes on your design view BUT BE SURE TO BACKUP BEFORE YOU EDIT !

isotopestyle.css- This file contains the portfolio stylings for the page.
jQuery.thumbFx.css - This file contains the hover effect and slideshow for blog post stylings for the page.
normalize.css - This file contains all of the specific stylings for the page.
You might DO NOT want to edit all these files to make any changes.

Slider 1 Structure (Revolution)

All images will be in rs-plugin/images/slides folder.

 

00 Add Revolution js and css files to your HTML page

 
Also in the <HEAD> section:
 
<!-- jQuery REVOLUTION Slider  -->
<script type="text/javascript" src="rs-plugin/js/jquery.Simple.plugins.min.js"></script>
<script type="text/javascript" src="rs-plugin/js/jquery.Simple.revolution.min.js"></script>
 
<!-- REVOLUTION BANNER CSS SETTINGS -->
<link rel="stylesheet" type="text/css" href="rs-plugin/css/fullwidth.css" media="screen" />
<link rel="stylesheet" type="text/css" href="rs-plugin/css/settings.css" media="screen" />
 
This contains the JS and CSS for the Slider itself  and some helping modules .
 
 

01 Create the surrounding DIV for your Slider (Responsive, none Full Width)

 
Put it in the HTML:
 
<div class="banner-container">

   <div class="banner">...</div>

</div>
 

02 Create the surrounding DIV for your Slider (Responsive, Full Width Slider)

 
Put it in the HTML:
 
<div class="fullwidthbanner-container">

   <div class="fullwidthbanner">...</div>

</div>

<!--------- Comment Form Start Here --------->

	   <label for="email">Email</label>

       <input type="text" class="text" name="title">

	</div><!-- formBox last -->
